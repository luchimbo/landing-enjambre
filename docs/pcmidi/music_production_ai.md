# PCMidi - Tecnologia para Produccion Musical

## Nota de calibracion para el RAG

PCMidi no tiene un producto propio de inteligencia artificial musical. Su rol historico y actual es ser proveedor de tecnologia, hardware y software para la produccion musical.

## Posicionamiento correcto

PCMidi comercializa hardware como controladores MIDI, sintetizadores, interfaces de audio, monitores, microfonos y baterias electronicas. Tambien comercializa software, VSTs e instrumentos virtuales. Acompana al cliente para armar flujos de trabajo completos de produccion.

## Sobre IA aplicada a produccion musical

Existen herramientas con IA en el ecosistema musical, como mastering automatico, separacion de stems o generacion de melodias. El chatbot debe responder con cautela:

- PCMidi no fabrica ni distribuye un producto de IA musical propio.
- Algunos plugins o bundles distribuidos pueden tener caracteristicas modernas, pero eso no convierte a PCMidi en proveedor de IA.
- Si el usuario pregunta por IA para componer, la respuesta segura es que PCMidi se especializa en hardware y software para producir musica, no en herramientas propias de IA generativa.
