# Launch checklist blog.pcmidicenter.com

## Estado

- Dominio final: `https://blog.pcmidicenter.com/`
- Hosting: Vercel
- Repo: `https://github.com/luchimbo/landing-enjambre`
- Sitemap: `https://blog.pcmidicenter.com/sitemap.xml`
- Robots: `https://blog.pcmidicenter.com/robots.txt`

## Verificado

- Home responde `200` por HTTPS.
- `sitemap.xml` responde `200` por HTTPS.
- `robots.txt` responde `200` y referencia el sitemap final.
- 500 URLs del sitemap responden y tienen canonical correcto.
- Assets principales `LogoPCMIDINegro.png` y `LogoPCMIDIBlanco.png` responden `200`.
- Una URL inexistente responde `404`.

## Pendiente cuando haya acceso a Google Search Console

1. Agregar propiedad para `blog.pcmidicenter.com`.
2. Verificar propiedad con DNS o metodo recomendado por Google.
3. Enviar sitemap: `https://blog.pcmidicenter.com/sitemap.xml`.
4. Revisar el reporte de indexacion despues de 24-72 horas.
5. Inspeccionar manualmente 5-10 URLs importantes y solicitar indexacion solo para las prioritarias.
